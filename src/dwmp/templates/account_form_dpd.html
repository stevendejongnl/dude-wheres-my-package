Connect DPD

DPD's parcels page is behind Cloudflare bot protection — there's no JSON API to call. You'll capture the rendered HTML from your logged-in browser session and paste it below. Re-capture whenever you need fresh data.

  1. Log in at dpdgroup.com/nl/mydpd/login.
  2. Navigate to My parcels → Incoming.
  3. Open DevTools (F12) → Console tab.
  4. Run this command: 
    copy(document.documentElement.outerHTML)
    Your clipboard now contains the page HTML.
  5. Paste it into the field below and click Test connection.
Strip surrounding quotes if your clipboard wrapped the HTML in a string literal.
