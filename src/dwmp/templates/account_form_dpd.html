{{ title_verb }} DPD

DPD's parcels page is behind Cloudflare — there's no JSON API. Paste your browser cookies as JSON and DWMP will launch a headless browser on every sync, reusing your session to fetch live data. Falls back to a frozen HTML snapshot if you prefer.

Recommended: cookies JSON (auto-refreshing)
  1. Log in at dpdgroup.com/nl/mydpd/login and open My parcels → Incoming so the Cloudflare challenge completes.
  2. Open DevTools (F12) → Console and run: 
    copy(JSON.stringify(document.cookie.split('; ').map(c => { const [name, ...v] = c.split('='); return {name, value: v.join('='), domain: location.hostname, path: '/'}; })))
    Click the snippet to copy it. Paste in the console and press Enter — cookies are now on your clipboard.
  3. Paste the cookies JSON in the field below.

For best results use the Cookie-Editor extension instead — it also captures httpOnly cookies that the console snippet can't access.

Fallback: HTML snapshot (one-shot, won't auto-refresh)
  1. Log in and open My parcels → Incoming.
  2. Open DevTools (F12) → Console and run: 
    copy(document.documentElement.outerHTML)
  3. Paste it below. You'll need to re-paste whenever you want new data.
DWMP detects the format automatically: starts with [ = cookies, starts with < = HTML.
In DevTools Console: copy(navigator.userAgent). Cloudflare binds cf_clearance to the UA that received it — a mismatch invalidates the cookie.
