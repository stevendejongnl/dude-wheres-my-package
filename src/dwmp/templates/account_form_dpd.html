Connect DPD

DPD's parcels page is behind Cloudflare — there's no JSON API. Paste your browser cookies as JSON and DWMP will launch a headless browser on every sync, reusing your session to fetch live data. Falls back to a frozen HTML snapshot if you prefer.

Recommended: cookies JSON (auto-refreshing)
  1. Install the Cookie-Editor browser extension.
  2. Log in at dpdgroup.com/nl/mydpd/login and open My parcels → Incoming so the Cloudflare challenge completes.
  3. Click Cookie-Editor → Export → JSON. Your clipboard now holds a JSON array.
  4. Paste it in the field below and click Test connection.
Fallback: HTML snapshot (one-shot, won't auto-refresh)
  1. Log in and open My parcels → Incoming.
  2. Open DevTools (F12) → Console.
  3. Run: 
    copy(document.documentElement.outerHTML)
  4. Paste it below. You'll need to re-paste whenever you want new data.
DWMP detects the format automatically: starts with [ = cookies, starts with < = HTML.
In DevTools Console: copy(navigator.userAgent). Cloudflare binds cf_clearance to the UA that received it — a mismatch invalidates the cookie.
