PostNL has no public API. You'll capture an OAuth token from your browser after logging in. Tokens are short-lived (~5 min) — the refresh token below keeps the session alive.

  1. Open jouw.postnl.nl/account/mijn-pakketten and log in.
  2. Open DevTools (F12) → Application tab → Session Storage → jouw.postnl.nl.
  3. Find the key: 
    akamai:1e450c3d-5bbb-4f34-9264-dd51fa9fd066:oidc-tokens
    Click the key name to copy it.
  4. Copy the access_token and refresh_token values from the JSON value.
  5. Paste them into the fields below and click Test connection.
Without this, you'll need to re-paste tokens every ~5 minutes.
